 `` Como usar meus dotfiles? ``
 ===========
 Usando o git:
~~~ sh
    git clone https://github.com/filipegmedeiros/dotsfiles.git # clone o repositório
    cd dotfiles/ # entre no diretório
    ln -s dotfiles/.config/polybar ~/.config/polybar # linke o repositorio
~~~
